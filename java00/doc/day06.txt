#배열

	==> 같은 형태(타입)의 데이터를 여러개 모아서 한꺼번에 관리하는 방식
	
		데이터가 많은 경우 원칙적으로 데이터 갯수만큼 변수를 만들어야한다.
		이러면 변수를 만들때도 불편하고 
		그 데이터를 사용할때도 엄청나게 불편하다
		
		이런 문제를 해결하기 위해서 제시된 방법이 배열이다.
		
		즉,
		배열이란?
		하나의 변수를 이용해서 여러개의 데이터를 관리할 수 있도록 하는것 
		
		
	참고] 
	
		자바에서 변수의 형태 
			1. 기본형 변수(Primitive type , Value 형태)
				==> 변수안의 기억된 내용이 데이터 자체인 경우
					변수를 찾으면 그 안에 사용가능한 데이터가 존재하는 것.
					
					8가지 데이터 타입으로 만들어진 변수 
					(boolean, char, byte, short, int , long , float, double) 
					
			2. 참조형 변수 ( Reference Type, 주소형태)
				==> 변수안의 기억된 내용이 주소인 경우 
					변수를 찾으면 그안의 데이터가 기어된 위치(주소)를 알려준다.
					
				기본타입 8가지 이외의 형태로 만들어진 모든 변수.
				
			참고]
				Heap 영역
					
					특징]
						1.반드시 주소를 알아야 사용할 수 있는 영역
						2.자바의 메모리 영역중 크기가 가장 큰 영역이며 확장이 가능한 역영이다 .
						  (부족하다면 스스로가 다른 영역을 빼앗아서 Heap 영역으로 사용할수 있다.)
						==> 그러므로 자바는 많은 양의 데이터가 필요하면 
							사이즈가 큰애들은 Heap영역에 기억하도록 되어있다.
							
						3.원칙적으로 한번 만들어지면 그 프로그램이 종료될때 까지 없어지지 않는 영역
						4. Heap 영역의 생기는 메모리는 자동 초기화 된다.
							(기본형변수는 자동초기화 된다.)
							
===================================================================================================================

배역 만드는 방법 
	1. 배열은 원칙적으로 데이터가 기억될 공간을 Heap 영역에 만들어야 한다.
	
		명령어 ] 
			
			new 데이터타입[ 개수 ]; 
			
				데이터타입 : Heap 에 만들 데이터 영역의 형태를 지정한다. (Heap Type)
				[ 개수 ] : Heap에 만들 데이터 영역의 개수를 지정한다.
				
			예] 
				new int[10];
					==> Heap 영역에 int 형 데이터를 기억할 메모리 공간은 10개 만들어 달라는 의미.
					
				new boolean[5] 
					==> Heap 영역에 boolean 형 데이터를 기역할 메모리 공간을 5개 만들어 달라는 의미.
					
			참고 ] 
				new 명령
				
					1. Heap에 뭔가를 만드세요 
					2. 만들어진 메모리에 시작 주소를 알려준다.
			
		
		2. Heap 에 있는 메모리의 주소를 기억할 변수를 만들어야 한다.
			
			명령어 ] 
				데이터타입[] 	변수이름; 
					예]
						int[] num; 
							
						데이터타입[] :	자신이 관리할 Heap 영역에 있는 
										실제 데이터 형태의 따라 달라진다.
										==> Heap Type 
										
						****
						결론]
							Heap Type이 같은 주소만 기억할수 있다.
							
					
						int[] num = new float[5]; ==> 안된다. Heap Type이 다르기 때문에
						
						float[] num = new int[5]; ==> 된다.  float가 int보다 크기때문에 가능하다.
						
				결론]
					배열 만드는 방법
						1. 만들 배열이 Heap에 생길 때 이 주소를 기억할 변수를 준비한다.
							
							int[] num;
							
						2. Heap에 필요한 메모리를 만들고 그 주소를 준비한 변수에 기억하면 된다.
						
							num = new int[5];
							
					배열 사용 방법
						
						명령어]
							
							변수[위치];
								
								참고 ]
									위치는 0부터 시작한다.
						
						참고]
							배열주소(변수).length
							==> 배열주소 관리하는 곳에 있는 메모리의 갯수를 알려주는 변수(명령).
							
							예]
								int[] num = new int[10];
								
								System.out.println(num.length);
								==> 10이 나온다 
								==> num이 가리키는곳에 가면 10개의 메모리가 존재하기 때문이다.
								
					참고]
						배열의 초기화
							Heap에 생기므로 자동으로 초기화가 이루어진다.
							(기본형 데이터 타입에 한해서 자동 초기화가 이루어진다)
							정수 - 0
							실수 - 0.0
							논리 - false
							문자 - 0(아스키코드값)		
						
						 경우에 따라서를 배열 메모리에 강제로 특정 값을 입력할 필요가 있을 것이다.
						 ==> 이때 이것을 배열의 초기화라고 한다.
						 
						 형식 1]
						 	int[] num = {1,2,3,4,5};
						 	
						 형식 2] 
						 	int[] num = new int[]{1,2,3,4,5};
						 	
						 
						 ***
						 주의]
						 	배열을 초기화 할때는 배열의 크기를 진정할수 없다. 
						 	왜냐하면 초기화 해놓는 데이터의 갯수로 배열의 크기가 정해진다.
						 	자동으로 배열의 크기가 결정이 된다.
						
					참고]
					
						Arrays,toString()
						==> 배열을 출력하기 위해서는 
							반복문을 사용해서 출력을 해야한다.
							하지만 단순히 배열의 내용을 확인만 하는 용도로는 조금 불편하다.
							
							따라서 배열의 기억된 데이터만 확인하는 용도로 
							배열의 가진 모든 데이터를 문자열로 변환하여 반환해주는 기능을 가진 함수가
								Arrays.toString()
							이다.
							
						형식]
							Arrays.toString(배열주소);

		참고]
			
			배열변수 선언 방법 2]
				
				데이터타입 	변수이름[];
				
				이방법도 "데이터타입[] 변수이름; " 과 똑같은 방법이다.
